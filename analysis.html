板块轮动判断

Sector Rotation Assessment

当前阶段：AI 主升浪重启 + 风险偏好回归（risk-on rotation back into growth）

过去一周 Mag7 因季末 rebalance + 涨价质疑承压，资金外流防御性板块；本日 GOOGL 入道指 + 美伊缓和共同触发资金回流。这是"再加仓"而非"扩散"。

资金流向：① 从 XLP/XLU 防御板块 → XLC/XLY/XLK 成长板块；② 从 XLE/XLB 商品板块 → 大型科技；③ 从 GLD 黄金 → BTC/SPY 风险资产；④ 从 USMV 低波 → MTUM/QQQ 动量。

下一阶段判断：关键观察 RSP/SPY 比值是否回升 — 若持续走平或回落，说明仍是 Mag7 主导的"窄宽度"行情；若 RSP/SPY 比值开始走高，则确认 broadening 启动，对中盘成长更友好。

风险点：QQQ RSI 65 接近 70 超买区，若继续连阳 2-3 天，可能出现技术性消化。
